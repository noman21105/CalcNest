Roman Numeral Rules

Basic Symbols:
I = 1, V = 5, X = 10, L = 50, C = 100, D = 500, M = 1000

Subtraction Rules:
I before V or X (IV = 4, IX = 9)
X before L or C (XL = 40, XC = 90)
C before D or M (CD = 400, CM = 900)
